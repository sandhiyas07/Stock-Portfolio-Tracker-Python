# 📈 Stock Portfolio Tracker

## 📖 Description

A simple Stock Portfolio Tracker developed using Python. This application allows users to enter a stock name and quantity, calculates the total investment value using predefined stock prices, and saves the portfolio details to a text file.

## ✨ Features

* 📊 Calculates total investment value
* 📝 Accepts stock name and quantity as input
* 💾 Saves portfolio details to a text file
* 🖥️ Console-based application
* ⚡ Simple and beginner-friendly

## 🛠️ Technologies Used

* Python

## 🚀 How to Run

1. Open the project in VS Code.
2. Run the Python file.
3. Enter the stock name and quantity.
4. View the total investment value.
5. Check the generated `portfolio.txt` file.
